    Introdução

    A validação de software é confirmação de que o produto é o desejado pelo usuário.

    Metodologia

    A estratégia escolhida para a elaboração desse documento é a comprovação informal, na qual contribuidores tentaram contactar a equipe de desenvolvedores e envolvidos na plataforma Duolingo para realizar a tarefa de leitura de descrições e uso do cliente para identificar possíveis problemas na expressão dos requisitos.
